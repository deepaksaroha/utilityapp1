# utilityapp1
A website which contains few of the most commonly used utility tools. 

## Utility Tools
- Text Encoder/Decoder
- RGB-HEX color code converter
- Time Stamp Converter
- Measuring Units Converter
- Number Base Converter
- Text Transformer

## Technologies used
- html
- css
- Javascript
- nodejs
- express ^4.17.1
- axios ^0.21.1
- Heroku
